# Contributing Guide

Thank you for helping build **Consent Token Protocol (CTP)** — part of Emotional Infrastructure™.

## Workflow
1. Fork the repo and create a branch: `feature/<short-name>`.
2. Keep PRs small and focused. Include tests in `tests/`.
3. Run checks locally: `pytest -q` and `flake8`.
4. Avoid adding telemetry. Consent is explicit, not implied.

## Commit Message Template
```
feat(issuer): short description
fix(server): short description
docs(docs): short description
```

## Code of Conduct
Respect survivors, protect dignity, and design for consent-first systems.
