# Security Policy

## Reporting a Vulnerability
Report issues privately to **security@emotionalinfrastructure.org**.
We will acknowledge within 48 hours and provide a remediation timeline.

## Supported Versions
| Version | Supported |
|--------:|:---------:|
| v0.2.x  | ✅ Active |
| v0.1.x  | ⚠️ Pilot only |

## Scope
This repo processes **emotional/behavioral consent signals**. No production PII should be submitted to issues or PRs. Use redaction.
